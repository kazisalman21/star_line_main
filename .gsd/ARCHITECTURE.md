## Conventions

- **Naming:** PascalCase for components, camelCase for utilities
- **Structure:** Feature-based pages, shared components, centralized data
- **Styling:** Tailwind utility classes + CSS custom properties
- **Animation:** GSAP for hero, CSS keyframes for ambient effects
- **State:** URL params for cross-page state; no global state manager
- **SEO:** `PageHead` component with `react-helmet-async` on each page
