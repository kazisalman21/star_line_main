## Overview

Starline Wayfinder is a premium intercity bus booking platform for **Star Line Group** (Bangladesh). It's a frontend-only React SPA with mock data, built using Lovable.dev. The app covers the full passenger journey: route search → seat selection → checkout → ticket confirmation, plus admin analytics and live tracking pages.

## Components

### AnimatedHero
- **Purpose:** Cinematic parallax hero section with 5 image layers that respond to mouse movement
- **Location:** `src/components/AnimatedHero.tsx`
- **Dependencies:** Framer Motion, SearchForm, hero layer images
- **Pattern:** Uses `useMotionValue` + `useSpring` + `useTransform` for smooth parallax

### SearchForm
- **Purpose:** Trip search form with From/To/Date/Passengers inputs
- **Location:** `src/components/SearchForm.tsx`
- **Dependencies:** React Router, mockData (cities), Lucide icons
- **Variants:** `hero` (glass card) and `compact` (plain card)

### Navbar
- **Purpose:** Fixed top navigation with desktop and mobile layouts
- **Location:** `src/components/Navbar.tsx`
- **Pattern:** AnimatePresence for mobile menu, active link highlighting via `useLocation`

### Data Layer (mockData.ts)
- **Purpose:** All application data — routes, bus results generator, bookings, admin stats
- **Location:** `src/data/mockData.ts`
- **Key exports:** `cities`, `popularRoutes`, `coachTypes`, `generateBusResults()`, `sampleBooking`, `adminStats`
- **Note:** `generateBusResults()` dynamically creates bus listings from route/coach data

## Data Flow

```
User Input → SearchForm → URL params → SearchResults page
                                          ↓
                                    generateBusResults()
                                          ↓
                                    Bus list rendered
                                          ↓
                                    SeatSelection → Checkout → TicketConfirmation
```

All data is generated client-side from `mockData.ts`. No API calls, no backend, no persistence.

## Design System

- **Fonts:** Plus Jakarta Sans (body), Space Grotesk (display/headings)
- **Color System:** HSL CSS variables with Shadcn/UI semantic tokens + custom `starline-red`, `starline-gold`, `starline-amber`
- **Animations:** Framer Motion for page transitions, Tailwind `tailwindcss-animate` for micro-interactions
- **Glass Cards:** Custom `.glass-card` and `.glass-card-elevated` utility classes
- **Dark Mode:** Class-based via `darkMode: ["class"]`

## Conventions

- **Naming:** PascalCase components, camelCase functions/variables
- **Imports:** Path aliases via `@/` prefix
- **State:** Local component state + URL search params (no global store)
- **Styling:** Tailwind CSS utility classes + Shadcn/UI design tokens

## Technical Debt

- [ ] No authentication system — all pages publicly accessible
- [ ] No data persistence — all state lost on refresh
- [ ] No error boundaries
- [ ] `useTransform` called inside `.map()` loop in AnimatedHero (React hooks rule violation)
- [ ] Hardcoded date "2026-03-25" in multiple places
- [ ] No loading/skeleton states for pages
- [ ] No SEO meta tags (title/description per page)
