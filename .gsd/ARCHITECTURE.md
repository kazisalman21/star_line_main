## Overview

Star Line is a **premium intercity bus booking platform** for Bangladesh, built as a single-page React application backed by Supabase (PostgreSQL + Auth + RLS). It supports full end-to-end booking flows, real-time tracking, admin fleet/route management, AI-powered customer care, and complaint management.
